• Main Function Syntax

static int Main(string[] args)
{
    //... 
    return 0;
}
The string array parameter that contains any command-line arguments passed in works just as in Java.
Thus, args[0] specifies the first command-line parameter, args[1] denotes the second parameter, and so on.

• Name Space syntax

namespace namespace_name
{
   // code declarations
}
The using keyword states that the program is using the names in the given namespace.
For example, we are using the System namespace in our programs.
